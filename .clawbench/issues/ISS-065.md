## Description
Zip-Slip path traversal in archive — `filepath.Walk` follows symlinks. Files discovered through symlinked directories pass the symlink check because the walk enters them transparently. A symlink pointing to `/etc` would include system files in the archive, and crafted entry paths with `../` could cause extraction to write outside the target directory.

## Impact
Sensitive system files can be included in downloaded archives. On extraction, crafted paths could write files outside the intended directory (Zip-Slip vulnerability).

## Suggestion
Use `filepath.WalkDir` instead of `filepath.Walk` (it does not follow symlinks by default). Add explicit symlink detection and skip symlinked directories. Validate each entry's path before including it in the archive to ensure it remains within the project root.
