## Description
HTML iframe sandboxed with both `allow-scripts` and `allow-same-origin` defeats sandbox protection. A sandboxed iframe with both permissions allows embedded content to remove its own sandbox via JavaScript.

## Impact
If HTML content comes from untrusted sources (user-uploaded files), malicious scripts can escape sandbox and access parent page context.

## Suggestion
Remove `allow-same-origin` unless there is a specific requirement. Sanitize HTML content before rendering or use a separate origin for the iframe.
