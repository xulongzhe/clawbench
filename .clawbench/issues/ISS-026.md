## Description
`ServeProjects` GET path leaks absolute filesystem paths in the response `path` field, while `ListDir` returns relative paths. This leaks server directory structure to the client.

## Impact
Information disclosure — server directory structure exposed to authenticated clients.

## Suggestion
Return a relative path consistent with `ListDir`, or compute `relFromBase` from `basePath`.
