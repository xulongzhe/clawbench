## Description

`nav.fileViewer` is referenced in App.vue (`t('nav.fileViewer')`) but does not exist in either en.ts or zh.ts locale files. The tab title for the file viewer will show the raw key path "nav.fileViewer" instead of a translated string.

## Impact

Visible UI bug — the file viewer tab title displays the raw i18n key instead of a localized label.

## Suggestion

Add `fileViewer: 'File Viewer'` to en.ts nav section and `fileViewer: '文件查看器'` to zh.ts nav section.
