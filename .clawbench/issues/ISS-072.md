## Description
No host key validation on load — there is a TOCTOU (Time-of-Check-to-Time-of-Use) race between checking the host key file's permissions and reading the key. An attacker with local access could replace the key file between the check and the read.

## Impact
A compromised host key allows MITM attacks on SSH connections. Users would connect to a spoofed server without warning.

## Suggestion
Open the file first with `os.Open`, then check the file descriptor's permissions via `os.File.Stat()`. Use the already-open file handle to read the key, eliminating the TOCTOU window. Alternatively, use `os.ReadFile` and check permissions in one syscall where possible.
