## Description
ChatPanel.vue has three missing features compared to ChatPanelContent.vue: (1) Missing `@show-thinking-detail` handler — thinking detail overlay never opens in BottomSheet mode. (2) Missing `thinkingEffort` from chat API request body — thinking effort configuration silently ignored. (3) Missing `onExtractScheduledTasks` callback.

## Impact
- Thinking detail overlay broken in BottomSheet mode
- Thinking effort selection has no effect in BottomSheet mode
- Scheduled task extraction broken in tab mode

## Suggestion
1. Add `@show-thinking-detail` handler
2. Add `thinkingEffort` to request body and ChatInputBar props
3. Add `onExtractScheduledTasks` to useChatStream options
