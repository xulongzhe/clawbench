## Description
`ServeLocalFile` uses `http.ServeFile` which follows symlinks and may serve files outside the project. TOCTOU race between validation and `ServeFile` could serve unintended files. `ServeFile` also serves `index.html` as directory listings.

## Impact
Potential arbitrary file read via symlink following or TOCTOU race.

## Suggestion
Use `http.ServeContent` with explicit `os.Open` + `defer Close` for all paths.
