## Description
TerminalPanel.vue and TerminalPanelContent.vue are ~90% duplicated (~800 lines each). The two files contain nearly identical code with minor differences, creating a maintenance burden where bug fixes and features must be applied twice.

## Impact
Divergent behavior between the two components over time as fixes are applied to one but not the other. Doubled maintenance effort for any terminal-related changes. Increased bundle size from duplicate code.

## Suggestion
Extract the shared logic into a composable (e.g., `useTerminalPanel`) or a base component. Have both components delegate to the shared implementation, providing only their specific differences. This ensures bug fixes propagate automatically and reduces code size.
