# ClawBench 界面功能测试用例

> 按功能模块组织，覆盖所有 UI 交互。测试时按模块顺序执行即可遍历全部功能。

---

## 前置条件

- 服务已启动（`./clawbench` 或 `./dev-server.sh`）
- 浏览器访问对应端口（生产 20000 / 开发 20001）
- 至少配置了一个 AI Agent（数据库中有智能体记录）
- 项目目录为 Git 仓库（部分 Git 测试需要）

---

## 一、登录与认证

### TC-AUTH-01 正常登录
1. 打开浏览器访问首页，应显示登录页
2. 输入正确密码，点击"提交"或按 Enter
3. **预期**：页面跳转至主界面，底部显示导航栏

### TC-AUTH-02 错误密码
1. 在登录页输入错误密码
2. 点击"提交"
3. **预期**：显示错误提示（带 X 图标），停留在登录页

### TC-AUTH-03 会话保持
1. 登录成功后刷新页面
2. **预期**：无需重新登录，直接进入主界面

### TC-AUTH-04 退出登录
1. 清除浏览器 Cookie（或使用无痕模式重新访问）
2. **预期**：跳转回登录页

2. 点击深色模式 / 浅色模式选项

## 四、聊天系统

### TC-CHAT-01 发送消息
1. 点击底部导航栏"聊天"按钮
2. 在输入框输入文本
3. 点击发送按钮或按 Enter
4. **预期**：消息发送成功，AI 开始流式响应，内容逐步显示

### TC-CHAT-02 流式响应显示
1. 发送消息后观察 AI 回复
2. **预期**：
   - 文字逐步出现（流式）
   - 思考块（Deep Thinking）可折叠展开
   - 工具调用（tool_use）显示图标+名称+状态（旋转/✓/✗）
   - 代码块有语法高亮
   - 数学公式用 KaTeX 渲染
   - Mermaid 图表正常渲染

### TC-CHAT-03 消息折叠/展开
1. 等待 AI 生成一条较长的回复
2. **预期**：长消息自动折叠，显示渐变遮罩 + "展开"按钮
3. 点击"展开"
4. **预期**：消息完全展开，底部出现"收起"链接
5. 点击"收起"
6. **预期**：消息重新折叠

### TC-CHAT-04 双击复制代码
1. 在 AI 回复中找到代码块
2. 双击代码块
3. **预期**：代码内容复制到剪贴板，有复制成功提示

### TC-CHAT-05 工具调用展开
1. 在消息中找到工具调用块（tool_use）
2. 点击工具调用块
3. **预期**：展开显示工具调用详情/输出结果
4. 再次点击
5. **预期**：折叠

### TC-CHAT-06 思考块折叠/展开
1. 在消息中找到"Deep Thinking"块
2. 点击标题栏
3. **预期**：思考内容折叠/展开切换

### TC-CHAT-07 警告/错误显示
1. 如遇到断连或超时
2. **预期**：显示黄色警告卡片或红色错误卡片

### TC-CHAT-08 停止生成
1. AI 正在生成回复时
2. 点击输入栏的停止按钮（一次）
3. **预期**：按钮变大 + 心跳动画（预备确认状态）
4. 再次点击（1.5 秒内）
5. **预期**：AI 停止生成，消息末尾显示"已取消"

### TC-CHAT-09 取消确认超时
1. AI 生成中，点击停止按钮一次
2. 等待超过 1.5 秒不再点击
3. **预期**：按钮恢复原状，不会取消

### TC-CHAT-10 消息排队
1. AI 正在生成时，在输入框输入新消息并点击发送
2. **预期**：消息显示为排队状态（橙色"收件箱"图标）
3. AI 完成当前回复后
4. **预期**：排队消息自动发送

### TC-CHAT-11 附件上传
1. 点击输入框左侧的回形针按钮
2. 在弹出菜单中选择"上传文件"
3. 选择一个文件上传
4. **预期**：附件标签出现在输入框上方，显示上传进度
5. 发送消息
6. **预期**：消息中显示附件文件标签

### TC-CHAT-12 拖拽上传
1. 从文件管理器拖拽文件到聊天输入区域
2. **预期**：出现拖放覆盖层
3. 释放文件
4. **预期**：文件上传，附件标签出现

### TC-CHAT-13 附加当前文件
1. 先在文件浏览中打开一个文件
2. 回到聊天，点击回形针按钮
3. 选择"当前文件"
4. **预期**：当前打开的文件作为附件标签出现

### TC-CHAT-14 引用最近文件
1. 点击回形针按钮
2. 选择"最近引用"下的一个文件
3. **预期**：文件作为附件标签出现

### TC-CHAT-15 移除附件
1. 添加附件后，点击附件标签上的 X 按钮
2. **预期**：附件被移除

### TC-CHAT-16 点击附件标签
1. 发送带有附件的消息
2. 点击消息中的附件标签
3. **预期**：在文件查看器中打开该文件

1. 在聊天消息区域左右滑动
2. **预期**：显示会话切换指示器（标题 + 位置圆点/胶囊滑块），切换到相邻会话

### TC-SESSION-04 创建新会话（短按）
1. 点击输入栏顶部的"+"按钮

3. 在 Agent 选择器中点击一个 Agent
4. **预期**：创建新会话并切换

### TC-SESSION-05 创建新会话（长按）
1. 长按"+"按钮 500ms
2. **预期**：直接使用默认 Agent 创建新会话（不弹出选择器）

### TC-SESSION-06 删除会话
1. 点击输入栏顶部的删除按钮
2. **预期**：弹出确认对话框
3. 确认删除
4. **预期**：会话被软删除，切换到其他会话

### TC-SESSION-07 未读标记
1. 有其他会话收到新消息时
2. 观察会话列表按钮
3. **预期**：按钮上显示未读数量徽章，带闪烁动画

### TC-SESSION-08 运行中标记
1. 有会话正在 AI 生成中
2. 观察会话列表按钮和会话列表
3. **预期**：按钮上显示旋转指示器，会话列表中对应项显示绿色圆点 + "运行中"

### TC-SESSION-09 加载更多历史消息
1. 打开一个消息较多的会话
2. 向上滚动到顶部
3. **预期**：显示加载指示器，加载更早的消息
4. 继续滚动直到全部加载
5. **预期**：显示"已全部加载"
